API

Endpoint: POST /API


Send a POST request with two image files to receive a verification score.


Example Request (Python):


import requests

url = "https://signature-verifier.ddns.net/API"
files = [
    ('sig1', open('image1.png', 'rb')),
    ('sig2', open('image2.png', 'rb'))
]
response = requests.post(url, files=files)
print(response.json())
